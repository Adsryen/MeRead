## 说明

- [x] 支持 RSS、Atom 格式
- [x] 导入和导出 OPML
- [x] 自定义字号、行间距、页边距、文本对齐方式
- [x] 三种阅读模式：阅读器、应用内标签页、系统浏览器
- [x] 未读筛选、文章收藏、订阅源分组
- [x] 文本选择同步到 [flomo](https://flomoapp.com/)
- [x] 适配深色模式
- [ ] 多语言支持
- [ ] 自定义 CSS
- [ ] 获取全文
